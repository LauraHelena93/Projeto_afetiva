# 🍞 Afetiva Pães e Doces — Website Institucional

Bem-vindo ao repositório do site da **Afetiva Pães e Doces**! Este é um projeto de página web institucional e responsiva desenvolvido para apresentar a marca, contar nossa história, exibir nossos produtos artesanais e facilitar o contato com os clientes.

---

## 🚀 Sobre o Projeto

O objetivo deste projeto é criar uma experiência digital acolhedora e profissional, refletindo a identidade da marca "Sabores que abraçam". O site funciona como uma vitrine virtual completa para os clientes conhecerem o cardápio e entrarem em contato.

### 📋 Funcionalidades Principais (Seções do Site)
* **Início / Home:** Apresentação da marca com a logo oficial e o slogan.
* **Nossa História:** Um espaço dedicado à trajetória da marca e ao carinho depositado em cada receita.
* **Nossos Produtos:** Exibição em carrossel/slides dos pães e doces artesanais (como pão caseiro, recheados e panetones).
* **Nossa Galeria:** Um mosaico organizado utilizando CSS Grid (com exibição em 3 colunas, bordas arredondadas e efeitos dinâmicos de hover).
* **Contato:** Links diretos e informações para pedidos e atendimento.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando práticas modernas de desenvolvimento web focado em UI/UX:

* **HTML5:** Estruturação semântica de todas as seções da página.
* **CSS3 & CSS Grid:** Estilização personalizada, criação de layouts responsivos, efeitos de transição (`transition`), zoom dinâmico (`scale`) e sombras projetadas (`box-shadow`).
* **W3.CSS:** Framework CSS leve utilizado para auxiliar na estrutura de containers e componentes responsivos (como o menu de navegação e os slides de produtos).

---

## 🎨 Demonstração do Design

O layout foi planejado para transmitir uma sensação de conforto e afeto, utilizando:
* Paleta de cores em tons terrosos e quentes (foco no tom `#4A2F18`).
* Tipografia limpa e elegante (`w3-serif`).
* Imagens com cantos arredondados (`border-radius: 35px`) para suavizar a interface.
